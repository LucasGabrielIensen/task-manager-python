# Task Manager em Python

## Descrição
Aplicação de terminal para gerenciamento de tarefas, desenvolvida em Python, com persistência de dados em arquivo JSON.

## Funcionalidades
- Criar tarefas
- Listar tarefas
- Marcar tarefas como concluídas
- Listar tarefas concluídas
- Remover tarefas
- Persistência de dados em JSON

## Tecnologias e conceitos
- Python
- Funções
- Listas e dicionários
- Manipulação de arquivos
- JSON
- Organização em módulos

## Como executar
```bash
python main.py
